# Manuais de Impressoras Zebra

Esta pasta contém os manuais das impressoras suportadas pelo sistema AXIS.

## Impressoras Suportadas

### ZT411 - Impressora Industrial
- **Tipo**: Impressora térmica de transferência industrial
- **Resolução**: 203 ou 300 dpi
- **Velocidade**: Até 6 polegadas/segundo
- **Conectividade**: Ethernet, USB, Serial

**Links para Download:**
- Manual do Usuário (PT): https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zt411-user-guide.html
- Manual Técnico: https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zt411-technical-specifications.html

### ZD421 - Impressora Desktop
- **Tipo**: Impressora térmica de transferência desktop
- **Resolução**: 203 dpi
- **Velocidade**: Até 6 polegadas/segundo
- **Conectividade**: Wi-Fi, Bluetooth, USB

**Links para Download:**
- Manual do Usuário (PT): https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zd421-user-guide.html
- Manual Técnico: https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zd421-technical-specifications.html

### ZQ630 - Impressora Pagewide
- **Tipo**: Impressora térmica de transferência pagewide
- **Resolução**: 300 dpi
- **Velocidade**: Até 8 polegadas/segundo
- **Conectividade**: Ethernet, Wi-Fi, USB

**Links para Download:**
- Manual do Usuário (PT): https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zq630-user-guide.html
- Manual Técnico: https://www.zebra.com/us/en/support-downloads/knowledge-articles/attachments/zq630-technical-specifications.html

## Como Adicionar Manuais Localmente

1. Baixe os PDFs dos manuais do site oficial da Zebra
2. Salve os arquivos nesta pasta com os nomes:
   - `ZT411-Manual-Usuario.pdf`
   - `ZD421-Manual-Usuario.pdf`
   - `ZQ630-Manual-Usuario.pdf`
3. O sistema irá detectar automaticamente e disponibilizar para download

## Nota

Os links acima direcionam para a página de suporte da Zebra onde você pode baixar os manuais oficiais em PDF.
